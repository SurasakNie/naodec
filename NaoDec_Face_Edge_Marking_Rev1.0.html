NaoDec — Face Edge Marking

The built structure is a regular dodecahedron with the bottom face left unbuilt — 11 pentagon faces, each a wooden frame with stretched fabric. Every shared edge between two faces carries the same letter, so assembly and tear-down only require matching identical letters — no numbering or lookup required. Edges with no letter (hatched in the diagrams below) aren't shared with another face at all — they rest directly on the platform/ground.

    Revision History

    • Rev 1.0 — Initial release. Pure structural framework (vertices, structural edges, faces) with the same v1–v20 node geometry as NaoDec_3D_Vertex_and_Edges_LED_Mapping_Rev1.3.html; all LED/electrical content (edge circuits, vertex LED daisy-chain, series coil, node 0 power-in, User POV/chair) lives in that file instead. The 30 structural edges carry the physical assembly letters (A–Z, no W) from the Face Edge Marking guides, derived from the anchors f1 = Top and f11 = Door; the 5 base-ring edges bordering the unbuilt bottom face f12 are OPEN. Vertices and node numbers are hidden by default (toggle back on via Display).
